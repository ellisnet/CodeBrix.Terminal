1. XtermSharp (https://github.com/migueldeicaza/XtermSharp)

The terminal engine code in the CodeBrix.Terminal.Engine namespace is a fork
of the XtermSharp library, which is an xterm-compatible terminal emulator
library for .NET, originally created by Miguel de Icaza. XtermSharp is itself
a port of the xterm.js library.

The CodeBrix.Terminal project includes code that has been copied, modified,
adapted, and/or repurposed from the XtermSharp library.

2. NStack (https://github.com/gui-cs/NStack) - Version 1.1.1

The Unicode text support code in the CodeBrix.Terminal.Text namespace is a fork
of the NStack library version 1.1.1. NStack is a .NET library that provides
Unicode string and rune (code point) support, ported from Go's string and rune
handling, with additional Unicode helper methods. NStack was originally created
by Miguel de Icaza and is maintained under the gui-cs organization.

The CodeBrix.Terminal project includes code that has been copied, modified,
adapted, and/or repurposed from the NStack library.
